The Opal Python SDK provides a comprehensive set of API classes to interact with the Opal Security platform programmatically. This reference covers all available API endpoints organized by category.

## Getting Started

All API classes are available through the `opal_security` package:

```python
from opal_security import UsersApi, GroupsApi, ResourcesApi
from opal_security import ApiClient, Configuration

# Configure the client
configuration = Configuration(
    api_key={'BearerAuth': 'your-api-key'}
)

# Create API instances
with ApiClient(configuration) as api_client:
    users_api = UsersApi(api_client)
    groups_api = GroupsApi(api_client)
    resources_api = ResourcesApi(api_client)
```

## Complete API Class List

| API Class | Description |
|-----------|-------------|
| **UsersApi** | Manage users, retrieve user details, and update user information |
| **GroupsApi** | Create, update, and manage groups and their memberships |
| **ResourcesApi** | Manage resources, resource access, and resource metadata |
| **RequestsApi** | Create access requests, approve/deny requests, and manage request lifecycle |
| **AppsApi** | Manage connected applications and app configurations |
| **AccessRulesApi** | Configure and manage automated access rules |
| **BundlesApi** | Create and manage access bundles containing groups and resources |
| **TagsApi** | Create, update, and manage tags for organizing resources |
| **OwnersApi** | Assign and manage resource owners |
| **GroupBindingsApi** | Configure group binding rules for automatic group assignments |
| **IdpGroupMappingsApi** | Map identity provider groups to Opal groups |
| **DelegationsApi** | Configure delegation policies for access management |
| **ConfigurationTemplatesApi** | Manage request configuration templates |
| **MessageChannelsApi** | Configure Slack and other notification channels |
| **OnCallSchedulesApi** | Integrate and manage on-call schedules from PagerDuty, Rootly, etc. |
| **SessionsApi** | Manage and audit user sessions |
| **EventsApi** | Query audit logs and system events |
| **UarsApi** | Create and manage User Access Reviews (compliance reviews) |
| **NonHumanIdentitiesApi** | Manage service accounts, bots, and other non-human identities |

## Core Classes

In addition to the API classes, the SDK provides essential core classes:

- **ApiClient** - HTTP client for making API requests
- **Configuration** - SDK configuration including authentication
- **ApiResponse** - Wrapper for API responses
- **OpenApiException** - Base exception class for API errors
