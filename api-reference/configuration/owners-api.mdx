The `OwnersApi` class provides methods to manage owners in Opal.

## Constructor

```python
from opal_security import ApiClient
from opal_security.api import OwnersApi

api_client = ApiClient()
api = OwnersApi(api_client)
```

<ParamField path="api_client" type="ApiClient" default="ApiClient.get_default()">
  Optional API client instance. If not provided, uses the default client.
</ParamField>

### set_owner_users

Sets the list of users for an owner.

<Info>
If escalation is enabled, the order of the user list determines the escalation priority order. If the owner has a source group, adding or removing users from this list will not be possible.
</Info>

```python
users = api.set_owner_users(
    owner_id="123e4567-e89b-12d3-a456-426614174000",
    user_id_list=user_ids
)
```

#### Parameters

<ParamField path="owner_id" type="UUID" required>
  The ID of the owner.
</ParamField>

## HTTP Response Variants

Each method has two additional variants for advanced use cases:

### with_http_info

Returns an `ApiResponse` object containing the response data, HTTP status code, and headers:

```python
response = api.get_owner_with_http_info(
    owner_id="123e4567-e89b-12d3-a456-426614174000"
)
print(response.data)  # The Owner object
print(response.status_code)  # HTTP status code
print(response.headers)  # Response headers
```

### without_preload_content

Returns the raw HTTP response without deserializing the content:

```python
raw_response = api.get_owner_without_preload_content(
    owner_id="123e4567-e89b-12d3-a456-426614174000"
)
```
