## Overview

The `BundlesApi` class provides methods for managing bundles in Opal. Bundles are collections of resources and groups that can be requested together. Use this API to create, retrieve, update, and delete bundles, as well as manage their associated resources and groups.

## Class Definition

```python
from opal_security.api import BundlesApi

api = BundlesApi()
```

## Bundle Management

### create_bundle

Creates a new bundle.

```python
create_bundle(
    create_bundle_info: CreateBundleInfo,
    _request_timeout: Optional[float] = None
) -> Bundle
```

<ParamField path="create_bundle_info" type="CreateBundleInfo" required>
  Information needed to create a bundle.
</ParamField>

<ParamField path="_request_timeout" type="float">
  Timeout setting for this request in seconds.
</ParamField>

**Returns:** `Bundle` - The created bundle object.

**Example:**

```python
from opal_security import Opal
from opal_security.models import CreateBundleInfo

client = Opal(api_key="YOUR_API_KEY")

bundle_info = CreateBundleInfo(
    name="Engineering Bundle",
    description="Resources for engineering team"
)

bundle = client.bundles.create_bundle(create_bundle_info=bundle_info)
print(f"Created bundle: {bundle.name}")
```

### get_bundles

Retrieves a paginated list of bundles.

```python
get_bundles(
    page_size: Optional[int] = None,
    cursor: Optional[str] = None,
    contains: Optional[str] = None,
    _request_timeout: Optional[float] = None
) -> PaginatedBundleList
```

<ParamField path="page_size" type="int">
  The maximum number of bundles to return. Default is 200, max is 1000.
</ParamField>

<ParamField path="cursor" type="str">
  A cursor indicating where to start fetching items after a specific point.
</ParamField>

<ParamField path="contains" type="str">
  A filter for the bundle name.
</ParamField>

<ParamField path="_request_timeout" type="float">
  Timeout setting for this request in seconds.
</ParamField>

**Returns:** `PaginatedBundleList` - A paginated list of Bundle objects.

**Example:**

```python
from opal_security import Opal

client = Opal(api_key="YOUR_API_KEY")

# Get first page of bundles
bundles = client.bundles.get_bundles(page_size=50)

for bundle in bundles.results:
    print(f"Bundle: {bundle.name}")

# Get next page
if bundles.next:
    next_bundles = client.bundles.get_bundles(cursor=bundles.next)
```

## Resource Management

### add_bundle_resource

Adds a resource to a bundle.

```python
add_bundle_resource(
    bundle_id: UUID,
    add_bundle_resource_request: Optional[AddBundleResourceRequest] = None,
    _request_timeout: Optional[float] = None
) -> BundleResource
```

<ParamField path="bundle_id" type="UUID" required>
  The ID of the bundle.
</ParamField>

<ParamField path="add_bundle_resource_request" type="AddBundleResourceRequest">
  The resource information to add to the bundle.
</ParamField>

<ParamField path="_request_timeout" type="float">
  Timeout setting for this request in seconds.
</ParamField>

**Returns:** `BundleResource` - The created bundle resource.

**Example:**

```python
from opal_security import Opal
from opal_security.models import AddBundleResourceRequest
from uuid import UUID

client = Opal(api_key="YOUR_API_KEY")

bundle_id = UUID("123e4567-e89b-12d3-a456-426614174000")
resource_id = UUID("987e6543-e21b-12d3-a456-426614174000")

request = AddBundleResourceRequest(
    resource_id=resource_id,
    access_level_remote_id="viewer"
)

bundle_resource = client.bundles.add_bundle_resource(
    bundle_id=bundle_id,
    add_bundle_resource_request=request
)
```
