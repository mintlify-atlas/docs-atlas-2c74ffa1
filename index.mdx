  <h2 className="text-3xl font-semibold dark:text-white text-gray-900 mb-4">
    Quick Start
  </h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with the Opal Python SDK in minutes
  </p>
  
  <Steps>
    <Step title="Install the package">
      Install the Opal Python SDK using pip:
      
      ```bash
      pip install opal-security
      ```
    </Step>
    
    <Step title="Configure authentication">
      Set up your API credentials to authenticate with the Opal API:
      
      ```python
      import os
      import opal_security as opal

      configuration = opal.Configuration(
          host="https://api.opal.dev/v1",
          access_token=os.environ["OPAL_TOKEN"]
      )
      ```
      
      <Note>
        Get your API token from the Opal dashboard under Settings → API Tokens
      </Note>
    </Step>
    
    <Step title="Make your first API call">
      Use the SDK to interact with Opal's API:
      
      ```python
      with opal.ApiClient(configuration) as api_client:
          # List all users
          users_api = opal.UsersApi(api_client)
          users = users_api.get_users()
          
          # List all groups
          groups_api = opal.GroupsApi(api_client)
          groups = groups_api.get_groups()
          
          print(f"Found {len(users.results)} users and {len(groups.results)} groups")
      ```
    </Step>
  </Steps>
</div>
