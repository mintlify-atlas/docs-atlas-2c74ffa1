This guide will help you install the Opal Python SDK and make your first API call.

## Installation

<Steps>
  <Step title="Install via pip">
    The recommended way to install the Opal SDK is from PyPI using pip:

    <CodeGroup>
    ```bash pip
    pip install opal-security
    ```

    ```bash pip (with sudo)
    sudo pip install opal-security
    ```
    </CodeGroup>

    <Note>
      The SDK requires Python 3.9 or higher. Check your Python version with `python --version`.
    </Note>
  </Step>

  <Step title="Alternative: Install from GitHub">
    You can also install directly from the GitHub repository:

    ```bash
    pip install git+https://github.com/opalsecurity/opal-python.git
    ```

    For development or to install from source:

    ```bash
    git clone https://github.com/opalsecurity/opal-python.git
    cd opal-python
    python setup.py install --user
    ```
  </Step>

  <Step title="Verify Installation">
    Confirm the SDK is installed correctly:

    ```python
    import opal_security as opal
    print(opal.__version__)
    # Output: 1.0.0
    ```
  </Step>
</Steps>

## Basic Usage

### Initialize the Client

First, import the SDK and configure authentication:

```python
import os
import opal_security as opal
from opal_security.rest import ApiException

# Configure the API client
configuration = opal.Configuration(
    host="https://api.opal.dev/v1",  # API endpoint (default)
    access_token=os.environ["OPAL_API_TOKEN"]  # Your API token
)

# Create an API client instance
api_client = opal.ApiClient(configuration)
```

<Warning>
  Never hardcode your API token in source code. Always use environment variables or a secrets management system.
</Warning>

### Make Your First API Call

Here's a simple example that lists all groups in your Opal organization:

```python
import opal_security as opal
import os
from pprint import pprint

# Configure authentication
configuration = opal.Configuration(
    access_token=os.environ["OPAL_API_TOKEN"]
)

# Use context manager for automatic cleanup
with opal.ApiClient(configuration) as api_client:
    # Create an instance of the Groups API
    groups_api = opal.GroupsApi(api_client)
    
    try:
        # Fetch all groups
        response = groups_api.get_groups()
        print("Groups in your organization:")
        pprint(response.dict())
    except ApiException as e:
        print(f"Exception when calling GroupsApi->get_groups: {e}")
```

### Working with Access Requests

Here's an example of creating an access request:

```python
import opal_security as opal
import os
from datetime import datetime, timedelta

configuration = opal.Configuration(
    access_token=os.environ["OPAL_API_TOKEN"]
)

with opal.ApiClient(configuration) as api_client:
    requests_api = opal.RequestsApi(api_client)
    
    # Create a new access request
    request_info = opal.CreateRequestInfo(
        requester_user_id="user-uuid-here",
        resources=[
            opal.CreateRequestInfoResourcesInner(
                resource_id="resource-uuid-here",
                access_level_remote_id="admin"
            )
        ],
        reason="Need access for project deployment",
        duration_minutes=480  # 8 hours
    )
    
    try:
        response = requests_api.create_request(request_info)
        print("Access request created successfully:")
        print(f"Request ID: {response.id}")
    except opal.ApiException as e:
        print(f"Error creating request: {e}")
```

### Query Events

Retrieve audit events with filtering:

```python
import opal_security as opal
import os

configuration = opal.Configuration(
    access_token=os.environ["OPAL_API_TOKEN"]
)

with opal.ApiClient(configuration) as api_client:
    events_api = opal.EventsApi(api_client)
    
    try:
        # Get events with filters
        response = events_api.events(
            start_date_filter="2024/01/01",
            end_date_filter="2024/12/31",
            page_size=100,
            cursor=None  # Use for pagination
        )
        
        for event in response.results:
            print(f"{event.event_type}: {event.actor} -> {event.object}")
            
    except opal.ApiException as e:
        print(f"Error fetching events: {e}")
```

## Common Patterns

### Using Context Managers

Always use the `with` statement to ensure proper resource cleanup:

```python
# Good: Context manager handles cleanup
with opal.ApiClient(configuration) as api_client:
    groups_api = opal.GroupsApi(api_client)
    groups = groups_api.get_groups()

# Avoid: Manual cleanup required
api_client = opal.ApiClient(configuration)
try:
    groups_api = opal.GroupsApi(api_client)
    groups = groups_api.get_groups()
finally:
    api_client.close()  # Must remember to close
```

### Error Handling

Handle API exceptions appropriately:

```python
from opal_security.rest import ApiException

try:
    response = groups_api.get_group(group_id="invalid-id")
except ApiException as e:
    print(f"Status code: {e.status}")
    print(f"Reason: {e.reason}")
    print(f"Body: {e.body}")
    
    # Handle specific error codes
    if e.status == 404:
        print("Group not found")
    elif e.status == 401:
        print("Invalid authentication credentials")
    elif e.status == 403:
        print("Access forbidden")
```

### Pagination

Many list endpoints support pagination:

```python
def fetch_all_groups(groups_api):
    """Fetch all groups using pagination."""
    all_groups = []
    cursor = None
    
    while True:
        response = groups_api.get_groups(
            cursor=cursor,
            page_size=200  # Default is 200
        )
        
        all_groups.extend(response.results)
        
        # Check if there are more pages
        if not response.next:
            break
            
        cursor = response.next
    
    return all_groups
```

## Type Hints and IDE Support

The SDK is fully typed with Pydantic models:

```python
from typing import List
import opal_security as opal

def get_admin_groups(api_client: opal.ApiClient) -> List[opal.Group]:
    """Get all groups with admin access level."""
    groups_api = opal.GroupsApi(api_client)
    response = groups_api.get_groups()
    
    # Type hints enable IDE autocomplete
    admin_groups: List[opal.Group] = [
        group for group in response.results
        if group.group_type == opal.GroupTypeEnum.ADMIN
    ]
    
    return admin_groups
```

<Note>
  The SDK uses Pydantic 2 for data validation. All models provide runtime type checking and helpful error messages for invalid data.
</Note>
