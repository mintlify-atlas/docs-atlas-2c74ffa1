The Opal Python SDK is a RESTful API client that allows you to interact with the Opal Security platform programmatically. Built with modern Python standards, this SDK provides a comprehensive interface for managing users, groups, resources, access requests, and more.

## What is Opal?

Opal is an access management platform that helps organizations automate and streamline their identity and access management workflows. The Python SDK enables developers to integrate Opal's capabilities directly into their applications and scripts.

## API Capabilities

The Opal Python SDK provides access to the following resources:

### Identity Management
- **Users**: Manage user accounts and their attributes
- **Groups**: Create and manage user groups with access levels
- **Non-Human Identities**: Handle service accounts and automation identities

### Access Control
- **Resources**: Manage resources across cloud providers (AWS, GCP, Azure) and SaaS applications
- **Bundles**: Group resources together for easier access management
- **Access Rules**: Define automated access policies
- **Requests**: Handle access request workflows and approvals

### Audit & Compliance
- **Events**: Track access events and changes
- **Sessions**: Monitor active user sessions
- **User Access Reviews (UARs)**: Conduct periodic access reviews

### Configuration
- **Message Channels**: Integrate with Slack and other messaging platforms
- **On-Call Schedules**: Connect with PagerDuty, Rootly, and other services
- **Configuration Templates**: Manage reusable configuration templates
- **Tags**: Organize resources with custom tags

## SDK Architecture

The SDK is organized into several key components:

```python
import opal_security as opal

# Configuration - API settings and authentication
configuration = opal.Configuration()

# ApiClient - HTTP client for API communication
api_client = opal.ApiClient(configuration)

# API Classes - Specialized clients for each resource type
groups_api = opal.GroupsApi(api_client)
resources_api = opal.ResourcesApi(api_client)
requests_api = opal.RequestsApi(api_client)

# Models - Pydantic data models for request/response validation
group = opal.CreateGroupInfo(name="Engineering", description="Engineering team")
```
